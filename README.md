# form_pollution_project

Proyecto monorepo que contiene dos subproyectos:

- `api/` — Backend REST construido con FastAPI (Air Quality API).
- `form-pollution-app/` — Frontend construido con React (Create React App).

Para documentación y pasos de arranque específicos de cada subproyecto, consulta sus README individuales:

- Backend (API): `api/README.md`
	- Accede a: ./api/README.md

- Frontend (UI): `form-pollution-app/README.md`
	- Accede a: ./form-pollution-app/README.md

Cómo usar este README:

- Si quieres levantar todo con Docker Compose, revisa `docker-compose.yml` en la raíz y sigue las instrucciones del README de `api/` y `form-pollution-app/`.
- Para desarrollo local, entra en la carpeta correspondiente y sigue el apartado "Instalación y arranque" de cada README.
